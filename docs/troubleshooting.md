# Troubleshooting

## Frontend build issues

If the frontend fails to build, confirm that the required environment variables are present and that Node.js dependencies were installed correctly.

## Contract test failures

If the Rust tests fail, verify that the correct Rust toolchain is installed and that the contract package dependencies are available.

## Wallet connection issues

Wallet-related problems are often caused by unsupported network settings, missing testnet configuration, or a misconfigured contract address.
