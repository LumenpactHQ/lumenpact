# Frequently Asked Questions

## What is Lumenpact?

Lumenpact is an open-source accountability platform built on Stellar. It lets users lock funds behind a commitment and rely on a judge to decide the outcome.

## Does it use zero-knowledge proofs today?

No. The current implementation uses a trusted-judge model for simplicity and accessibility.

## Is this intended for real funds?

The repository is primarily intended for testnet exploration and development. Any real-fund deployment should be reviewed carefully and follow the security guidance in this repository.

## Can I contribute?

Yes. Please read [CONTRIBUTING.md](CONTRIBUTING.md) and open an issue or pull request with your ideas.
