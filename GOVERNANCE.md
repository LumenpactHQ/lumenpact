# Governance

Lumenpact is currently maintained by a small core group of contributors responsible for architecture, review, and release coordination.

## Roles

- Maintainers: review pull requests, manage releases, and guide technical direction.
- Contributors: propose improvements, submit patches, and participate in issue triage.
- Community members: report issues, ask questions, and help with documentation.

## Decision making

Decisions are made by consensus where possible. For significant changes that affect security, contract behavior, or user-facing product direction, maintainers should seek review from at least one additional maintainer before merging.

## Changes to this document

Proposed updates to governance should be discussed in an issue or pull request before they are adopted.
